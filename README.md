# Game Art Guidebook

React + TypeScript application for a multi-user game art guidebook with Supabase-backed auth and content management.

## Development

Requirements:

- Node.js
- npm

Start locally:

```sh
npm install
npm run dev
```

Build for production:

```sh
npm run build
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
